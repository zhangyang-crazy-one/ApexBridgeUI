Plugin System Features

  • Modal Overlay - Full-screen backdrop with backdrop-filter: blur(4px) for depth
  • HTML & IFrame Loading - Support for both HTML string content and external URL iframes
  • ESC Key Dismissal - Press ESC to close the active plugin modal
  • Click Outside to Close - Click the backdrop to dismiss the modal
  • Smooth Animations - Scale (0.9‚Üí1.0) + fade transitions at 300ms
  • Z-index Management - Plugin modal sits at z-index: 9999
  • Singleton Pattern - Only one plugin active at a time for focus
  • Custom Sizing - Specify custom width and height per plugin
  • postMessage API - Secure communication with iframe plugins
  • Custom Events - Dispatches plugin-mounted and plugin-unmounted events
  • Full Accessibility - Proper ARIA roles (role="dialog", aria-modal="true")
  • Theme Aware - All colors adapt to light/dark theme switching
