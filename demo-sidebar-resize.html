Sidebar Resize Demo

This demo showcases the implementation of CORE-006 (resizable left sidebar) and CORE-007 (resizable right sidebar) using the Anthropic design system.

Drag & Drop

Grab the vertical handles between sidebars and main content to resize. Smooth 300ms transitions follow Anthropic design principles.

Min/Max Constraints

Sidebars can resize between 180px and 400px. Default width is 260px for optimal usability.

localStorage Persistence

Sidebar widths are saved to localStorage with keys vcpchat-sidebar-left-width and vcpchat-sidebar-right-width.

Touch Support

Works seamlessly on mobile and tablet devices with touch events in addition to mouse drag.

Visual Feedback

Resize handles show hover (gray) and active (black) states using Anthropic color variables.

Theme Aware

All colors adapt smoothly when switching between light and dark themes. Try the toggle button above!

Try It Out

  1. Hover over the vertical resize handles (they'll turn gray)
  2. Click and drag to resize the sidebars horizontally
  3. Release to save the new width to localStorage
  4. Reload the page - your widths will be restored!
