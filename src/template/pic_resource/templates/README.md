## 概述

本原型模板系统是DAMA数据管理平台的前端UI/UX设计参考实现,基于DMBOK2(Data Management Body of Knowledge v2)标准,展示了企业级数据管理平台的核心功能和用户体验设计。

### 核心目标

1. **展示理解深度**: 通过可交互的原型展示对数据建模、元数据管理、血缘追踪等专业领域的深入理解
2. **验证设计理念**: 验证UI/UX设计方案的可行性和用户体验
3. **快速迭代**: 使用原生HTML/CSS/JavaScript实现,便于快速预览和修改
4. **设计参考**: 为最终的React/Vue实现提供设计规范和交互参考

---

## 设计理念

### 1. 专业性 (Professional)

- **企业级设计规范**: 采用简洁、专业的蓝灰色调配色方案
- **一致性**: 统一的组件样式、交互模式和视觉语言
- **DMBOK2对齐**: 界面设计体现数据管理最佳实践和工作流程

### 2. 易用性 (Usability)

- **直观导航**: 清晰的左侧导航栏 + 顶部工具栏布局
- **信息架构**: 合理的信息层级和内容组织
- **交互反馈**: 即时的视觉反馈和状态提示

### 3. 可访问性 (Accessibility)

- **键盘导航**: 支持Tab键导航和Enter键确认
- **语义化HTML**: 使用正确的HTML标签和ARIA属性
- **对比度**: 文字和背景色对比度符合WCAG 2.1 AA标准

### 4. 响应式 (Responsive)

- **桌面优先**: 针对桌面应用优化(最小1024x768)
- **Flexbox/Grid布局**: 灵活的布局系统适配不同窗口大小
- **缩放友好**: 支持浏览器缩放(80%-150%)

## 功能页面

### 1. 登录页面 (index.html) - NEW: Anthropic风格重设计

**功能**: DAMA平台的登录页面和功能展示

**设计风格**: 基于Anthropic.com的设计理念
- **配色方案**: 温暖中性色(米白背景#FAF9F5 + 深黑文字#141413)
- **字体系统**: 标题无衬线 + 正文衬线,大字号(20px正文)
- **极简主义**: 黑白灰配色,无彩色点缀
- **充足留白**: 大间距,呼吸感强

**核心特性**:
- **导航栏**: DAMA Logo(黑色版本) + 语言切换 + 帮助
- **Hero区域**: 大标题 + 副标题 + 使命说明
- **登录表单**: 居中卡片,简洁输入框,黑色按钮
- **功能展示**: 像Anthropic一样用文字清晰说明6个核心功能
- **响应式设计**: 适配桌面和移动设备

**交互**:
- 登录表单提交(模拟)
- 功能标题可点击跳转到对应页面
- 链接悬停效果

**DMBOK2对齐**:
- 清晰展示数据建模、元数据管理、血缘追踪等核心功能
- 体现企业级数据管理平台的专业性和可信赖感

### 2. 数据建模页面 (modeling.html)

**功能**: 可视化的ERD编辑器和DDL生成工具

**核心特性**:
- **三层模型设计**: 支持概念/逻辑/物理模型切换
- **拖拽式编辑器**: 从左侧实体模板拖拽到画布
- **实时DDL生成**: 选择实体后自动生成DDL脚本
- **多平台支持**: PostgreSQL/MySQL/Oracle/SQL Server

**交互流程**:
1. 选择模型类型(概念/逻辑/物理)
2. 从左侧拖拽实体模板到画布
3. 点击实体查看属性详情
4. 右侧面板显示DDL预览
5. 选择目标平台生成对应DDL

**技术亮点**:
- 使用原生JavaScript实现拖拽功能
- 动态生成DDL代码(语法高亮)
- 实体节点可拖动重新布局

**DMBOK2对齐**:
- 符合三层模型架构(概念/逻辑/物理)
- 主键/外键标识清晰
- 支持多平台DDL生成

### 6. 数据流矩阵页面 (data-flow-matrix.html)

**功能**: 二维矩阵展示企业内部各系统/应用之间的数据流动关系

**核心特性**:
- **DMBOK2标准矩阵**: 符合DMBOK2第108页的数据流矩阵设计规范
- **业务流程箭头**: 顶部展示业务流程阶段(采集→清洗→整合→分析→应用→归档)
- **矩阵热力图**: X轴和Y轴为系统列表,单元格显示数据流关系
- **数据流标识**: 大圆点(●)表示创建数据,小圆点(●)表示查询/使用数据
- **详情面板**: 点击单元格查看数据流规格(频率、数据量、涉及表等)

**交互流程**:
1. 查看顶部业务流程箭头了解数据流向
2. 浏览矩阵单元格,圆点表示数据流关系
3. 点击单元格查看详细的数据流规格
4. 使用筛选器按数据域、流向类型、频率筛选
5. 导出Excel或生成AI分析报告

**技术亮点**:
- 原生JavaScript动态渲染矩阵
- 黑白配色,专业简洁
- 响应式表格布局
- 单元格点击交互

**DMBOK2对齐**:
- 数据架构管理 (Data Architecture Management)
- 数据集成与互操作性 (Data Integration & Interoperability)
- 应用交互矩阵 (Application Interaction Matrix)

## 技术实现

### 技术栈

- **HTML5**: 语义化标签,ARIA属性
- **CSS3**: Flexbox/Grid布局,CSS变量,动画
- **JavaScript (ES6+)**: 原生JavaScript,无框架依赖
- **D3.js v7**: 数据可视化(ERD图、血缘图)

### 设计系统

#### 配色方案

```css
/* 主色调 */
--primary-color: #2563eb;      /* 蓝色 - 主要操作 */
--primary-hover: #1d4ed8;      /* 深蓝 - Hover状态 */
--primary-light: #dbeafe;      /* 浅蓝 - 背景高亮 */

/* 辅助色 */
--secondary-color: #64748b;    /* 灰色 - 次要信息 */
--success-color: #10b981;      /* 绿色 - 成功状态 */
--warning-color: #f59e0b;      /* 橙色 - 警告状态 */
--danger-color: #ef4444;       /* 红色 - 危险状态 */

/* 中性色 */
--bg-primary: #ffffff;         /* 白色 - 主背景 */
--bg-secondary: #f8fafc;       /* 浅灰 - 次背景 */
--text-primary: #0f172a;       /* 深灰 - 主文字 */
--text-secondary: #475569;     /* 中灰 - 次文字 */
```

#### 间距系统

```css
--spacing-xs: 0.25rem;   /* 4px */
--spacing-sm: 0.5rem;    /* 8px */
--spacing-md: 1rem;      /* 16px */
--spacing-lg: 1.5rem;    /* 24px */
--spacing-xl: 2rem;      /* 32px */
```

#### 字体系统

```css
--font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, sans-serif;
--font-size-xs: 0.75rem;   /* 12px */
--font-size-sm: 0.875rem;  /* 14px */
--font-size-base: 1rem;    /* 16px */
--font-size-lg: 1.125rem;  /* 18px */
--font-size-xl: 1.25rem;   /* 20px */
```

## 使用方法

### 本地预览

1. **直接打开**: 在浏览器中打开 `index.html` 文件
2. **本地服务器** (推荐):
   ```bash
   # 使用Python
   cd frontend/src/templates
   python -m http.server 8000
   
   # 使用Node.js
   npx http-server -p 8000
   
   # 访问 http://localhost:8000
   ```

### 导航流程

1. 打开 `index.html` 查看原型概览
2. 点击任意功能卡片进入对应页面
3. 使用左侧导航栏在不同页面间切换
4. 体验各页面的交互功能

### 修改和扩展

1. **修改样式**: 编辑 `styles/common.css` 中的CSS变量
2. **添加功能**: 在对应HTML文件的 `<script>` 标签中添加JavaScript
3. **新增页面**: 复制现有页面模板,修改内容和交互逻辑

## 设计规范

### 布局规范

- **侧边栏宽度**: 240px (固定)
- **顶部栏高度**: 60px (固定)
- **内容区域**: 自适应剩余空间
- **最小窗口**: 1024x768

### 交互规范

- **按钮点击**: 150ms过渡动画
- **Hover效果**: 背景色变化 + 阴影提升
- **加载状态**: 显示加载指示器(typing indicator)
- **错误提示**: 使用模态框或Toast提示

### 可访问性规范

- **键盘导航**: Tab键切换焦点,Enter键确认
- **焦点样式**: 蓝色边框 + 阴影
- **ARIA标签**: 为交互元素添加 `aria-label`
- **对比度**: 文字对比度 ≥ 4.5:1
